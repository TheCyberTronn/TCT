Okay so because I don't want to belabour the point, the following tags exist:

  • UL to make this list work.
  • LI to make this list have all the list entries
  • ADDRESS for when you need to write an address
  • CITE for citating things.
  • BDO to make text reverse

  • KBD, SAMP, and CODE make it look like computer code.
  • VAR for scientific variables
